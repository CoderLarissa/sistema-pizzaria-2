<h1 align="center">🍕 Fictional Pizza shop management system </h1>
<p align="center"> Web Systems Course Project </p>

<h1>Description</h1>
<p>This project was developed for the Web Systems course and simulates the management of a pizza shop. The platform allows administrators to log in and perform CRUD operations on customers and pizza flavors, as well as simulate pizza orders.

The ordering process is based on a common real-world scenario in which customers place their orders through WhatsApp and the administrator registers them in the system. The project was created to practice frontend development concepts using React and modern web technologies.

</p>

<h1> Setup and Installation</h1>
<h2>Prerequisites</h2>

<p>Before running this project, make sure you have installed:</p>

<ul>
  <li>Node.js (LTS version recommended)</li>
  <li>npm (comes with Node.js)</li>
</ul>

<p>You can check if they are installed by running:</p>

<pre>
    node -v
    npm -v
</pre>

<hr>

<h2>Clone the Repository</h2>

<p>First, clone this project to your local machine:</p>

<pre>
git clone https://github.com/CoderLarissa/sistema-pizzaria-2.git
</pre>

<p>Then navigate into the project folder:</p>

<pre>
    cd your-repository-name
</pre>

<hr>

<h2>Install Dependencies</h2>

<p>This project uses React, React Router and Bootstrap. Install all dependencies by running:</p>

<pre>
    npm install
</pre>

<p>This will install:</p>
<ul>
  <li>React and React DOM</li>
  <li>React Router DOM (for navigation and routes)</li>
  <li>Bootstrap (for styling and layout)</li>
</ul>

<hr>
<h2>Run the Project</h2>
<p>To start the development server, run:</p>

<pre>
    npm run dev
</pre>

<p>After that, open the URL shown in the terminal (usually http://localhost:5173/ if using Vite).</p>

<hr>
<h2>Important Notes</h2>
<ul>
  <li>This project uses React Router for navigation between pages.</li>
  <li>Make sure to always run the project from the root directory.</li>
  <li>If Bootstrap styles are not applied, ensure it is properly imported in <code>main.jsx</code> or <code>index.js</code>.</li>
</ul>
